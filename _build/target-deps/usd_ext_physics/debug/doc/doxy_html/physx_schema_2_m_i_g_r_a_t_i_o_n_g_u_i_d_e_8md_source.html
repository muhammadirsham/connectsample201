1 PhysxSchema Migration Guide
2 ===========================
3 
4 May 2021
5 ----------
6 * PhysxMeshCollisionAPI was removed. It was replaced by PhysxConvexHullCollisionAPI, PhysxConvexDecompositionCollisionAPI and
7 PhysxTriangleMeshSimplificationCollisionAPI. Based on the UsdPhysicsMeshCollisionAPI:approximation attribute the matching schema
8 is used.
9 * PhysxCookingDataAPI was changed to multiple applied schema, the schema token defines which cooked data are stored. This allows us to store multiple versions of cooked data per mesh, for example convex approximation for a dynamic collision and triangle mesh for a static collision.
10 
11 March 2021
12 ----------
13 
14 * General:
15  - For vehicles, many IsA schemas have been converted to API schemas. This allows for a more compact (fewer prims) vehicle setup for cases where there is no intent to share vehicle components among vehicle instances. For details, see further below.
16 * PhysxVehicleGlobalSettings:
17  - Was renamed to PhysxSchemaPhysxVehicleContextAPI and is now an API schema.
18  - In addition, this API has to be applied to a UsdPhysicsScene prim now.
19 * PhysxVehicleWheel:
20  - Was renamed to PhysxSchemaPhysxVehicleWheelAPI and is now an API schema.
21  - Optionally, this API can be applied directly to the prim that has PhysxSchemaPhysxVehicleWheelAttachmentAPI applied instead of using the physxVehicleWheelAttachment:wheel relationship
22 * PhysxVehicleTire:
23  - Was renamed to PhysxSchemaPhysxVehicleTireAPI and is now an API schema.
24  - Optionally, this API can be applied directly to the prim that has PhysxSchemaPhysxVehicleWheelAttachmentAPI applied instead of using the physxVehicleWheelAttachment:tire relationship
25 * PhysxVehicleSuspension:
26  - Was renamed to PhysxSchemaPhysxVehicleSuspensionAPI and is now an API schema.
27  - Optionally, this API can be applied directly to the prim that has PhysxSchemaPhysxVehicleWheelAttachmentAPI applied instead of using the physxVehicleWheelAttachment:suspension relationship
28 * PhysxVehicleEngine:
