Detailed Description

CharacterControllerAPI can be applied to a capsuleGeom.

It will turn the capsule into a character controller.

Applies this single-apply API schema to the given prim.

This information is stored by adding "CharacterControllerAPI" to the token-valued, listOp metadata apiSchemas on the prim.

Returns
A valid PhysxSchemaCharacterControllerAPI object is returned upon success. An invalid (or empty) PhysxSchemaCharacterControllerAPI object is returned upon failure. See UsdPrim::ApplyAPI() for conditions resulting in failure.
